Installation instructions For Mac: Download and install TextWrangler - http://www.barebones.com/products/textwrangler/ Git (choose the one for your operating system) http://code.google.com/p/git-osx-installer Anaconda Scientfic python distribution https://store.continuum.io/ Testing 1) Open a terminal window (it is in Applications/Utilities) type git as long as you don’t get the message -bash: git: command not found You are set! 2) In your terminal window type ipython notebook If an ipython dashboard window opens in your browser, you’re set! Feel free to try out some commands or simple addition. Shift+Enter executes the contents of the cell. -------------------------------------------------------------------------------------------------------------------------- For Windows: Download and install Notepad++ http://notepad-plus-plus.org/download/v6.4.3.html git bash https://openhatch.org/missions/windows-setup/install-git-bash Anaconda Scientfic python distribution https://store.continuum.io/ Testing 1) Open git bash (NOT git-gui) as described in https://openhatch.org/missions/windows-setup/open-git-bash-prompt) If you get a window that looks like this: http://www.prosoftstudio.com/wp-content/uploads/2011/05/01-Open-Git-Bash.png You’re set! NOTE: To copy/paste into/out of this window, right click on the top bar of the window, go to the properties menu, and click on ‘QuickEditMode’. Once you have selected that, highlight text and then hit enter to copy, right-click to paste. 2) in that window type: ipython notebook If an ipython dashboard window opens in your browser, you’re set! Feel free to try out some commands or simple addition. Shift+Enter executes the contents of the cell.
